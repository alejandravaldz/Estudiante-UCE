# 📘 Proyecto: Registro de Calificaciones en C#

## 📌 Descripción
Este programa en C# permite:
- Registrar las calificaciones de varios estudiantes.
- Almacenarlas en una lista.
- Calcular el promedio general.
- Imprimir en pantalla el listado con **nombre y calificación**, junto con el promedio final.

---

## 🚀 Funcionalidades
- Entrada de datos (nombre y calificación).
- Almacenamiento dinámico en una lista.
- Cálculo automático del promedio.
- Visualización clara de resultados.

---

## 🛠️ Tecnologías utilizadas
- Lenguaje: **C#**
- Entorno: **.NET Console Application**

---

## 📂 Estructura del código
- `Program.cs`: Contiene el método `Main` y la lógica principal.
- `List<string>` y `List<decimal>`: Para almacenar nombres y calificaciones.
- Métodos auxiliares para:
  - Registrar datos.
  - Calcular promedio.
  - Mostrar resultados.
